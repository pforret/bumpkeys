# bumpkeys

Upgrade your SSH keys for better security

## Installation

with [basher](https://github.com/basherpm/basher)

	$ basher install pforret/bumpkeys

or with `git`

	$ git clone https://github.com/pforret/bumpkeys.git
	$ cd bumpkeys

## Usage

	> bumpkeys.sh [action]
